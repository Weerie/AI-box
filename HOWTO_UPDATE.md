# How to Update Weerie's AI Box

At the moment, updating **Weerie's AI Box** requires a full reinstall.

## Update steps

1. **Uninstall the current version**
   - Run the uninstaller located in the application directory.

2. **Keep your local chats and agents (optional)**
   - During uninstall, make sure to enable the checkbox for **"Save local chats / agents"** if you want to keep your existing data.
   - You can find the storage location of these files in the **Personal Settings** section of the app.

3. **Download the latest installer**
   - Get the newest installer from the **Releases** page.

4. **Install the new version**
   - Run the installer and install the app again in the folder of your choice.

## Notes

This update process is currently the only supported method.

It is not ideal and may feel confusing or unfriendly. This will remain the case until the update system is reworked.
